# QuadTree Spatial Indexing System (C++)

## 📌 Project Description

This project implements a QuadTree-based spatial indexing system in C++.
It efficiently stores and queries location data for real-world applications such as ride-sharing, restaurant search, social media, and emergency services.

## ✨ Features

* Insert and manage spatial locations
* Search location by name or ID
* Count locations within a region
* Query locations in a specific area
* Find nearest location (e.g., nearest cab)
* Find K nearest neighbors
* Filter locations by type (CAB, RESTAURANT, USER, HOSPITAL)
* Display QuadTree structure

## 🌍 Real-World Applications

* 🚕 Ride-sharing (Find nearest cab)
* 🍔 Food delivery (Find nearby restaurants)
* 📱 Social media (Nearby friends)
* 🚑 Emergency systems (Nearest hospital)

## 🛠️ Technologies Used

* C++ Programming
* Data Structures (QuadTree)
* Recursion
* STL (vector, algorithm)
* Object-Oriented Programming (OOP)
